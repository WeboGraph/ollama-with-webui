
# Ollama + OpenWebUI

Simple package to install and manage Ollama and OpenWebUI.

## Installation

```sh
chmod u+rx *.sh 

```

## Start

```sh
./start.sh
```

start and update

```sh
./update.sh
```

## Stop

```sh
./stop.sh
```
